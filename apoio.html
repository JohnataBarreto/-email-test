Olá Aspirante,
Este é um email marketing e o seu teste consiste em desenvolve-lo, aplicando as melhores práticas para este tipo de peça. Cuidado: Para email, precisamos voltar aos primórdios de diagramação Web (tabelas). Uma vez que os clients de email travam diversas tags e estilos globais. Isso para proteção do webmail em geral, portanto é necessário ter atenção e pesquisar as melhores práticas (além das que pontuamos aqui). Faça e envie: Crie um repositório público no Github e envie para o email: contato@shiftinc.com.br com o título: ”Teste de email”. Critérios de avaliação:
✔️ HTML semântico e boas práticas em geral
✔️ Uso de CSS inline
✔️ Melhor aproveitamento de uso de tabelas. Evitar exageros. Utilização de colspan e rolspan - se necessário.
✔️ Teste visual do email nos principais clientes: Gmail, Outlook, Mail, Yahoo, ...
